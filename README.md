# LiSPER Dashboard

Static public status for the LiSPER binding-free-energy and Li/Na selectivity
study. All content is in `index.html`; there is no backend or build step.

Preview with:

```bash
python3 -m http.server 8080
```

The dashboard reports only current audited inputs, declared estimands, active
method validation, and the number of completed affinity/selectivity rows.
